# Calculadora API - ASP.NET Core

![C#](https://img.shields.io/badge/C%23-239120?style=flat&logo=c-sharp&logoColor=white)
![ASP.NET Core](https://img.shields.io/badge/ASP.NET_Core-512BD4?style=flat&logo=dotnet&logoColor=white)

Projeto simples de **calculadora** desenvolvido em **ASP.NET Core (C#)**. Esta é minha **primeira API**, criada para praticar o desenvolvimento de **APIs REST**.

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão  
- Recebe valores via URL e retorna o resultado

## Tecnologias

- C#  
- ASP.NET Core  
- API REST

## Como Executar

1. Clone o repositório:  
   ```bash
   git clone <URL_DO_REPOSITORIO>
2. Abra a solução no Visual Studio
3. Execute o projeto
4. Acesse a API no navegador ou com ferramentas como Postman

## Objetivo
Praticar a criação de APIs REST e consolidar conceitos de C# e programação orientada a objetos.
